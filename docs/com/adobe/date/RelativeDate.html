getRelativeDate()method
public static function getRelativeDate(firstDate:Date, secondDate:Date = null, capitalizeFirstLetter:Boolean = false):String

Creates a human-readable String representing the difference in time from the first date provided with respect to the second date provided. If no second date is provided, then the relative date will be calcluated with respect to "now". This method handles dates in both the past and the future (e.g. "2 hours ago" and "2 hours from now". For any date beyond 3 days difference from now, then a standard format is returned.

Parameters

firstDate:Date — The date for which to compare against.
 
secondDate:Date (default = null) — The date to use as "present" when comparing against firstDate.
 
capitalizeFirstLetter:Boolean (default = false)

Returns
String — Human-readable String representing the time elapsed.
